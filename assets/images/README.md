# Image Files

Add the following `.png` files to this folder:

- `room-shift.png`
- `door-memory.png`
- `sound-before-thought.png`
- `answer-next-click.png`
- `background-moving.png`
- `sentence-chose-you.png`
- `quiet-louder.png`
- `happened-before.png`

These filenames map directly to the question `graphic` values in `src/questions.js`.
Draw and motion parameters live in `src/background.js` under `GRAPHIC_SETTINGS`.
